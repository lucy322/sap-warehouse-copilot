🤖 SAP Warehouse Copilot

Reachy Mini × NVIDIA NIM × Riva Speech AI × SAP OData

Voice-driven inventory assistant that queries SAP data through natural language.

Features

  • 🎤 NVIDIA Riva ASR — Speak to your robot, speech transcribed in real-time
  • 🧠 NVIDIA NIM LLM — Llama 3.1 70B with SAP function-calling tools
  • 🔊 NVIDIA Riva TTS — Robot speaks responses back naturally
  • 📦 SAP OData Integration — Stock levels, POs, maintenance orders, KPIs
  • 🤖 Expressive Robot — Head nods for healthy, shakes for critical, droops for out-of-stock
  • 📊 Live Dashboard — Real-time KPI cards, tool-call trace, conversation log

Quick Start

  1. Set your NVIDIA_API_KEY (get free credits at build.nvidia.com)
  2. Install: pip install -e .
  3. Run: reachy-mini-daemon
  4. Open dashboard at http://localhost:7860
